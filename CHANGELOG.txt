## Changelog for psychopy-whisper plugin

# psychopy-whisper 0.0.6 (2025-02-12)

- Use the `openai-whisper` package for transcription instead of `faster-whisper`
- Added 'auto' device selection for torch for backwards compatibility

# psychopy-whisper 0.0.3 (2023-11-15)

- Added some ignored arguments to WhisperTranscriber.transcribe() to be in keeping with the base class (which was written for Sphinx and Google transcribers)

# psychopy-whisper 0.0.1 (2023-07-13)

First basic version
